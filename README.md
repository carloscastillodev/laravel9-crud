# CRUD básico con Laravel 9 y Laraver UI

Creación de un CRUD (Create, Read, Update, Delete) usando Laravel 9 y Resource Controllers. Además, se implementa la funcionalidad de eliminación lógica de registros (Soft Delete) y la confirmación antes de eliminar un registro usando jQuery y SweetAlert2.
